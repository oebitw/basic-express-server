## Setup

1) Clone the repo
2) npm init -y.
3) npm install.
4) Create .env file with PORT=3030.

## Test
* npm test


## Run the app
* npm start

## End points

* `/person`

You can add any name using:
* `/person?name=omar`

return object :
```
{
   "name": "omar"
}
```
